# mp-bit-trees-maven

A mini-project exploring bit trees (a form of binary tree) and their use in translating between alphabets, particularly in translating to and from braille.

**Instructions for use**

In order to use this, first launch the Braille ascii.java function, and as run parameters when calling the function give a first parameter of the type of output you want the function to give, and the string of charachters you give as input.
